### SPA vs MPA 권장
**SPA (Single Page Application)** 권장 — 풀스크린 섹션 스크롤 방식
- 브랜드 사이트 특성상 스토리텔링 흐름이 중요
- 섹션 전환 시 부드러운 트랜지션 효과
- 각 섹션을 앵커 내비게이션으로 연결

### 3.2 Philosophy (브랜드 철학)

| 항목 | 내용 |
|------|------|
| 레이아웃 | 스크롤 트리거 애니메이션, 텍스트 중심 |
| 구조 | 3단계 — 미션 → 달항아리 스토리 → 3가지 가치 |

**미션 섹션**
```
"We Craft Time, Not Just Scent."
우리는 향을 통해 복잡한 세상의 스위치를 끄고,
오롯이 당신 내면과 마주하는 '사유의 시간'을 설계합니다.
```

**달항아리 스토리**
- 비주얼: 달항아리 클로즈업 이미지/영상
- 텍스트: 위아래를 따로 만들어 붙이는 과정, 이음새의 미학
- 인터랙션: 스크롤에 따라 달항아리가 천천히 회전하거나 조명이 변하는 효과

**3가지 핵심 가치** (카드 또는 순차 등장)

| 번호 | 가치 | 키워드 | 설명 |
|------|------|--------|------|
| 01 | Deep Layering | 깊은 겹 | 시간의 흐름에 따라 겹겹이 쌓이는 잔향의 미학 |
| 02 | Imperfect Beauty | 비정형 | 사람의 손길이 닿은 자연스러운 일그러짐 |
| 03 | Ultimate Void | 여백 | 비워냄으로써 완성되는 한국적 여백의 미 |

## 4. 디자인 시스템

### 디자인 디렉션
- **럭셔리 패션매거진** 같은 에디토리얼 레이아웃
- 깨끗한 **흰 바탕** + 라이트 브라운 포인트
- **여백이 핵심** — 콘텐츠보다 여백이 훨씬 넓은 편집적 구성
- 글과 사진의 간격, 여백이 디자인의 완성도를 결정
- 이미지와 텍스트가 겹쳐지는 **불규칙 그리드** 허용
- 얇은 세리프 서체로 고급스러움 강조

### 이미지 운용
| 유형 | 설명 | 플레이스홀더 |
|------|------|-------------|
| Background | 와이드 또는 세로형, 섹션 배경으로 깔리는 대형 이미지 | 라이트 브라운 영역 |
| Thumbnail | 제품/디테일 컷, 편집적 배치 | 라이트 브라운 영역 |
| Video | 분위기 영상, 공간 영상 | 라이트 브라운 영역 |

> 이미지/동영상 영역은 우선 라이트 브라운(`#D4C4B0`) 영역으로 자리만 잡아둔다.

### 컬러 팔레트

| 용도 | 컬러 | 코드 |
|------|------|------|
| Background (Primary) | White | `#FFFFFF` |
| Background (Warm) | Ivory | `#FAF8F5` |
| Image Placeholder | Light Brown | `#D4C4B0` |
| Text (Primary) | Near Black | `#1A1A1A` |
| Text (Secondary) | Warm Gray | `#8A8078` |
| Text (Muted) | Light Gray | `#B8B0A8` |
| Accent | Warm Brown | `#C4A882` |
| Border | Subtle | `rgba(0,0,0,0.08)` |

### 타이포그래피

| 용도 | 폰트 | Weight | 비고 |
|------|------|--------|------|
| Display (EN) | Cormorant Garamond | 300, 400 | 얇은 세리프, 럭셔리 |
| Heading (EN) | Playfair Display | 300, 400 | 에디토리얼 강조용 |
| Body (KR) | Noto Serif KR | 200, 300 | 얇은 명조, 고급 |
| Body (EN) | Cormorant | 300, 400 | 라이트한 세리프 |
| UI/Caption | Pretendard | 300 | 가독성 보조 |

### 레이아웃 원칙
1. **여백 > 콘텐츠** — 화면의 60% 이상이 여백
2. **불규칙 그리드** — 이미지와 텍스트가 겹치고 오프셋 배치
3. **에디토리얼 편집** — 패션매거진처럼 사진과 타이포가 조화
4. **얇은 선, 얇은 글씨** — 무게감 없이 가벼운 인상

### 인터랙션 원칙
1. **느림의 미학** — 모든 트랜지션은 0.8s~1.5s, ease-out 곡선
2. **여백 우선** — 콘텐츠보다 여백이 더 넓은 레이아웃
3. **자연스러운 등장** — 급격한 전환 없이 서서히 나타남
4. **최소한의 UI** — 네비게이션, 버튼은 극도로 절제
